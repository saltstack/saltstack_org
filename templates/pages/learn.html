Learn & Communicate

Salt benefits from a vibrant and growing community. Our goal is to build a community where we learn from each other and where we can reach our goals together.

Joining the community is easy, join the Salt development on GitHub, chat with the team in IRC, and get updates and join in-depth discussions on the mailing list.

Learn the code

Join the Salt development team on GitHub, contribute code, watch the active development, fork Salt Stack projects and help us deliver a continually growing and exciting project. If you are new to git and GitHub there are many tools to help you become involved in Salt.

Fork Salt on GitHub and get involved!

Salt’s code is hosted on GitHub, if you are unfamiliar with git and github, then there are many resources available. Look at the github help to learn how to contribute code to Salt!

http://help.github.com

IRC

Chat and ask questions in the #salt channel on Freenode. Get to know the community!

New to IRC? IRC Tips

Search before you ask
  • Mailing list archives
  • IRC logs
No introductions needed
There's no need to introduce yourself or ask "Can I ask a question?" Just ask!
Wait for an answer
If no one responds right away, don't give up. The person with the answer might be busy coding or afk. Leave your window open and someone will reply to you.
Use the webchat interface
Use Freenode's webchat interface to connect without installing any software.
